      Method

      Point-on-serve probability for server i against returner j is computed via the Barnett–Clarke formulation, pij = fi + (1 − f̄) − gj, where f is service-points-won, g is return-points-won, and f̄ is tour-average SPW on the chosen surface.

      Game, set, tiebreak, and match probabilities follow from exact Markov-chain recursion; the Monte Carlo column is an independent check using a seeded Mulberry32 generator.

      Data

      Populate the CSV with values from the ATP stats leaderboard, filterable by timeframe and surface. Columns: player, surface, period, spw, rpw. Surfaces: all · hard · clay · grass. Periods: 52week · career.

      The recency weight blends w·52-week + (1−w)·career. A standard setting is 0.70.
